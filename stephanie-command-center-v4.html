Step 1 — Set up CallMeBot (2 minutes)
1

Save CallMeBot as a WhatsApp contact

Add this number to your WhatsApp contacts: +34 644 59 71 91
Name it anything, like "CallMeBot".

2

Send the activation message

Open WhatsApp, message that number exactly this:

I allow callmebot to send me messages

You'll get a reply with your API key. Copy it — you'll need it in Make.com later.

Step 2 — Create your Google Apps Script
1

Open Google Apps Script

Go to script.google.com and click New project. Delete everything in the editor.

2

Paste this code

Copy the entire block below and paste it into the script editor:

const SHEET_NAME = 'Deadlines'; function doPost(e) { try { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.appendRow(['ID', 'Date', 'Text', 'Pillar', 'LastUpdated']); } // Clear and rewrite all deadlines const lastRow = sheet.getLastRow(); if (lastRow > 1) sheet.deleteRows(2, lastRow - 1); data.deadlines.forEach(d => { sheet.appendRow([d.id, d.date, d.text, d.pillar, new Date().toISOString()]); }); return ContentService .createTextOutput(JSON.stringify({ success: true, count: data.deadlines.length })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ success: false, error: err.message })) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) return ContentService.createTextOutput(JSON.stringify({ deadlines: [] })).setMimeType(ContentService.MimeType.JSON); const rows = sheet.getDataRange().getValues(); const deadlines = rows.slice(1).map(r => ({ id: r[0], date: r[1], text: r[2], pillar: r[3] })); return ContentService .createTextOutput(JSON.stringify({ deadlines })) .setMimeType(ContentService.MimeType.JSON); }
3

Create a Google Sheet and link it

Go to sheets.google.com, create a new blank sheet. Copy its URL. Back in Apps Script, click Resources → Advanced Google Services or just go to Project Settings and note your script ID. Then in the script editor click the title and rename it "Command Center Sync".

4

Link the script to your Sheet

In the Apps Script editor, click Deploy → New deployment. Set type to Web app. Set "Execute as" to Me and "Who has access" to Anyone. Click Deploy, authorize it, and copy the Web app URL. Paste that URL into the field at the top of this page.

Step 3 — Set up Make.com automation
1

Create a free Make.com account

Go to make.com and sign up free. No card needed.

2

Create a new scenario

Click Create a new scenario. Add these three modules in order:

Module 1: Schedule — set to run every day at 8:00 AM (WAT = UTC+1, so set to 07:00 UTC).

Module 2: HTTP → Make a request — set URL to your Google Apps Script URL (same one you pasted above). Method: GET.

Module 3: HTTP → Make a request — this sends the WhatsApp message via CallMeBot.

3

Configure the WhatsApp message module

In Module 3, set:

URL: https://api.callmebot.com/whatsapp.php Method: GET Query string: phone = +2349011261732 text = [your message — see below] apikey = [your CallMeBot API key from Step 1]

For the text field, use Make.com's text builder to loop through the deadlines from Module 2 and build a message like:

Good morning Stephanie! 🌅 Today's briefing: {{deadlines due today or in 3 days}} You've got this.
4

Turn it on

Click the Scheduling toggle to ON. Your scenario will now run every morning at 8am, check your deadlines, and send you a WhatsApp message automatically.

Once this is set up: every time you add or edit a deadline in your command center, it automatically syncs to your Google Sheet. Make.com reads the sheet every morning and sends your WhatsApp briefing. You never have to touch the Sheet directly.
